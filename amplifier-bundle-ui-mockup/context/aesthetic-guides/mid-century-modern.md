# Mid-Century Modern Cozy - Aesthetic Guide

**Proven palette from meditation app mockup session (14/14 successful mockups)**

---

## Color Palette

### Primary Colors
```css
--terracotta: #d4745f;      /* Warm, inviting primary */
--olive-green: #7a9b76;     /* Natural, grounding secondary */
--mustard-yellow: #e8b44f;  /* Energy, optimism accent */
```

### Neutrals
```css
--cream-bg: #f9f6f0;        /* Warm background, never pure white */
--warm-charcoal: #3a3633;   /* Text, never pure black */
--soft-coral: #f4a692;      /* Optional lighter accent */
```

---

## Typography

### Font Families
**Headers/Titles:** Serif fonts for warmth and personality
- Georgia
- Crimson Text
- Merriweather
- Playfair Display

**Body Text:** Sans-serif for clarity and readability
- Inter
- SF Pro / System UI
- Open Sans

### Hierarchy
- **Large headers:** Serif, warm charcoal
- **Body text:** Sans-serif, medium gray
- **Labels/captions:** Small serif or sans, olive/terracotta accents
- **Pull quotes:** Large serif, terracotta color

### The Mix
Combining serif (personality) + sans-serif (clarity) creates the mid-century literary magazine feeling.

---

## Shapes & Forms

### Rounded Corners
- **Cards/containers:** 16-24px radius
- **Buttons:** 20-32px radius (pill shapes)
- **Modals:** 24-32px radius (top corners for sheets)

### Organic Shapes
- **Not perfect circles** - Slightly irregular, hand-drawn feeling
- **Not rigid rectangles** - Always rounded, never 90° corners
- **Natural forms** - Think kidney shapes, pebbles, organic furniture

### Visual Rhythm
- Vary sizes (not uniform grid)
- Overlapping elements (depth through layering)
- Asymmetric balance (not centered perfection)

---

## Spacing & Layout

### Generous Whitespace
- **Between major sections:** 32-48px
- **Between cards:** 16-24px
- **Inside cards:** 16-24px padding
- **Margins:** 24-32px on mobile sides

### Visual Breathing Room
- Don't pack the canvas
- Let content have space around it
- Alternate dense and sparse areas

---

## Shadows & Depth

### Soft Shadows
```css
box-shadow: 0 4px 12px rgba(58, 54, 51, 0.08);  /* Warm shadow */
```
- Never harsh black shadows
- Warm-toned shadows (charcoal with low opacity)
- Subtle depth, not dramatic 3D

### Glows (Active Elements)
```css
box-shadow: 0 0 20px rgba(212, 116, 95, 0.3);  /* Terracotta glow */
```
- Use for primary actions (voice button, CTAs)
- Warm color glow matching element

## When to Use This Aesthetic

**Best for:**
- Personal productivity tools
- Creative applications
- Contemplative interfaces
- Reading/writing apps
- Knowledge management
- Meditation/wellness apps

**Not ideal for:**
- Enterprise dashboards (too casual)
- Financial tools (need more serious tone)
- Gaming interfaces (need more energy)
- E-commerce (need more urgency)

---

## Common Mistakes to Avoid

❌ **Pure white backgrounds** → Use cream #f9f6f0  
❌ **Perfect geometric shapes** → Add organic irregularity  
❌ **All sans-serif typography** → Mix with serif for warmth  
❌ **Black text on white** → Use charcoal #3a3633 on cream  
❌ **Harsh shadows** → Use warm-toned soft shadows  
❌ **Tight spacing** → Give generous breathing room  
❌ **Cold accent colors** → Stick to warm earth tones
